# TujenMem

ExileAPI Plugin for Path of Exile - Tujen Haggle Automation

## Features

- **Automated Haggling**: Automatically negotiate prices with Tujen
- **Price Integration**: Uses poe.ninja prices for item valuation
- **Custom Pricing**: Set custom prices for specific items
- **Haggle History**: Track all haggle sessions with detailed statistics
- **Price Display**: View all downloaded prices from poe.ninja
- **Session Analytics**: 
  - Total profit tracking
  - Artifact consumption tracking
  - Reroll cost calculation
  - Item purchase history

## Installation

1. Download the latest release
2. Extract to your ExileAPI Plugins folder
3. Restart ExileAPI

## Configuration

### Settings

- **League**: Select your current PoE league
- **Artifact Value Settings**: Configure chaos values for each artifact type
- **Custom Prices**: Override poe.ninja prices for specific items
- **Blacklist/Whitelist**: Filter items to haggle

### Haggle History

Track detailed statistics for each haggle session:
- Total rolls and coins spent
- Exotic Coinage price at time of session
- Artifact usage (Lesser, Greater, Grand, Exceptional)
- Items bought with stack amounts
- Total profit including reroll costs

## Usage

1. Download price data from poe.ninja
2. Configure your settings
3. Open Tujen's window in-game
4. Start the haggle process
5. View history and statistics
